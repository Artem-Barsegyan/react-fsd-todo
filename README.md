## react-fsd-todo

**Это мой первый проект, построенный по методологии Feature-Sliced Design.**  
Todo-приложение с папками задач, написанное на React + TypeScript.

### Технологии

- React + TypeScript
- Feature-Sliced Design — архитектурная методология
- React Context — управление состоянием
- React Router — навигация
- LocalStorage — персистентность данных
- CSS Modules — стилизация

### Возможности

- Папки задач — создавай папки с цветовой индикацией
- Управление задачами — добавляй, удаляй, отмечай выполненные
- Редактирование — меняй названия папок прямо на месте
- Адаптивный дизайн — мобильная версия с бургер-меню
- Доступность — полная поддержка клавиатуры (ARIA-метки, навигация по Enter)
- Автосохранение — все задачи сохраняются в localStorage
